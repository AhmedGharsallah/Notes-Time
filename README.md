# Notes-Time

## Description
This application allows users to create, manage, and organize their notes efficiently.

## Features
- Create, edit, and delete notes.
- Organize notes into categories.
- Tag notes for easier searching.
- Search functionality.
- Export notes to different formats.

## PyInstaller Build Instructions
To build the application into an executable, use PyInstaller. Make sure you have PyInstaller installed:
```bash
pip install pyinstaller
```
Then, navigate to the directory containing `main.py` and run:
```bash
pyinstaller --onefile main.py
```
This will create an executable in the `dist` folder.

## Run Command
To run the application, use the following command:
```bash
python main.py
```
